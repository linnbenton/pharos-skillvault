# Pharos SkillVault 🛡️

SkillVault adalah backend engine berbasis TypeScript untuk mengelola aset, distribusi kas (treasury), serta analisis risiko keamanan pada jaringan **Pharos Atlantic Testnet**.

## Spesifikasi Jaringan

- **Nama Jaringan:** Atlantic Testnet
- **Chain ID:** 688689
- **Currency:** PHRS
- **Explorer:** https://atlantic.pharosscan.xyz/

## Cara Memulai

```bash
npm install
npm run deploy
npm start
```
