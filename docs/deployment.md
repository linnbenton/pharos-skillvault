# Atlantic Testnet Deployment Guide

1. Run `npm install` to install all required dependencies.
2. Copy `.env.example` to `.env` and fill in your `PRIVATE_KEY` loaded with PHRS Faucet tokens.
3. Run `npm run deploy` to test the RPC connectivity (`https://atlantic.dplabs-internal.com`).
4. Run `npm start` to fire up the live API server.
